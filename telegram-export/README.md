# Экспорт из Telegram @KlimovoZ

Сюда положите **полный экспорт канала** — я сам отберу фото продукции и разложу по карточкам.

## Как экспортировать

**Telegram Desktop** (рекомендуется):

1. Откройте канал [@KlimovoZ](https://t.me/KlimovoZ)
2. **⋮** → **Export chat history**
3. Формат: **JSON** (удобно для подписей) или **HTML**
4. Медиа: включить **Photos** (и **Videos**, если есть ролики с продуктами)
5. Сохраните папку сюда целиком, например:
   ```
   telegram-export/KlimovoZ-export/
   ```

Можно просто скопировать папку `photos/` из экспорта — тоже подойдёт.
