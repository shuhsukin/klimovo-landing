# Климово — лендинг

Лендинг молочной фермы в деревне Климово (Старицкий район, Тверская область).

**Сайт:** https://shuhsukin.github.io/klimovo-landing/

## Локально

```bash
open index.html
```

Или Live Server в Cursor.

## Деплой

Репозиторий: `git@github.com:shuhsukin/klimovo-landing.git`  
GitHub Pages: ветка `main`, корень `/`.

```bash
git add .
git commit -m "описание изменений"
git push origin main
```

Обновление на сайте — обычно 1–2 минуты после push.
